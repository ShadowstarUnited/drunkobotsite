About



Drunk o'Bot is a unique kind of Fun-Bot for Discord. While simulating a general purpose moderation bot, Drunk o'Bot does not actually kick or ban anybody. Instead, it just has random funny and snarky responses to every command!

The idea behind Drunk o'Bot came to me legitimately as an early morning "what if" thought. I was laid in bed; had just made a coffee and I was thinking to myself: "What if there was a Discord Bot that did NOT do what you wanted it to do?". And so I messaged a friend about my idea

Not only has it been incredibly fun to create Drunk o'Bot, but it has also helped me overcome a lot of depression. I would spend long hours each day just working on Drunk o'Bot, and it would distract me from my woes; as well as improving my skills as a developer.
Seeing Drunk o'Bot grow as quickly as it did has been such an amazing experience for me, and the feedback I have received from the many Server Owners who invited the bot has given me the motivation to continue to work on it. Thank you very much <3

- Shadowstar#7815
Developer

The current commands are as followed:

- warn (user): This command has Drunk o'Bot send random funny responses whenever you try to warn someone. Note that Drunk o'Bot does not actually warn people.

- mute (user): This command has Drunk o'Bot send random funny responses whenever you try to mute someone. Note that Drunk o'Bot does not actually mute people.

- kick (user): This command has Drunk o'Bot send random funny responses whenever you try to kick someone. Note that Drunk o'Bot does not actually kick people.

- ban (user): This command has Drunk o'Bot send random funny responses whenever you try to ban someone. Note that Drunk o'Bot does not actually ban people.

- bean (user): This command has Drunk o'Bot send random funny responses whenever you try to bean someone.

- flirt (user): This command has Drunk o'Bot flirt with someone on your behalf.

- givecookie (user): This command lets you give a cookie to someone.

- fortune: This command posts funny Fortune Cookie quotes.

- topic: This command helps revive 'ded chat' by posting a conversation starting topic for your members to talk about! There are over 300!

- wisdom: This command has Drunk o'Bot send some funny advice. (Do not actually take his advice.)

- joke: This command has Drunk o'Bot send a funny beer related joke such as 'man walked into a bar'.

- ask: This is your classic Magic 8-Ball command, but with aggressive drunk answers!

- slot: This is a shitty currently-does-nothing-good command.

- activity: This command is only usable by Server Admins, and is used to set activity messages in a channel. Once a channel is set, Drunk o'Bot will send a message for every 40 messages sent in that channel, and the members must be the quickest to react to the correct kind of message!

- help: This is Drunk o'Bot's help menu.

- ping: This command might display the bot's latency, but don't count on a drunk to be accurate.

- botinfo: This command tells you some stats about Drunk o'Bot, such as how many beers he has drank.

- clear: Perfectly emulates everyone's favorite blue modbot's clear message command. (Note: It's totally useless.)

- invite: Invite Drunk o'Bot to join your underdeveloped server and steal your booze.

- donate: Tells you to donate to a Charity instead. No, really. I don't want your money.

- suggest: Suggest new features for Drunk o'Bot!

- support: This command lets you join our Support Server.

- privacy: This command displays our Privacy Policy.

- prefix: This command display's the bot's current prefix. Server Admins may set a custom prefix with `d!prefix set (new prefix)`.
